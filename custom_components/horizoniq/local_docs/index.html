HorizonIQ

Local documentation served from your Home Assistant instance.

This integration polls your HorizonIQ forecast API, exposes forecast entities in Home Assistant, and caches tokens needed for efficient polling.

Configuration Fields

FieldRequiredDescription
home_assistant_installation_idRead-onlyStable Home Assistant installation identifier from core.uuid, shown for copying when registering licensing or trials. Not stored by this integration.
urlYesForecast API endpoint URL.
api_keyYesAPI key sent in request header X-API-KEY.
battery_capacity_sensorYesEntity ID whose state is sent as currentBatteryCapacity.
environmentNoEnvironment selector (Live or Sandbox).
hashNoDeterministic result of the most recent forecast, sent as the hash query value.
registration_dataNoRegistration data for your site. Sent as registrationData so it is not constantly reloaded from the database. Refreshes daily or when force-refreshed with the clear registration button.
forecast_device_idNoCanonical device ID for app-owned trial binding, sent as X-HorizonIQ-Device-Id when set. Victron/Home Assistant setups usually use the GX device ID.
forecast_device_tokenNoPortal-generated app trial device token, sent as X-HorizonIQ-Device-Token when set.

Entities

Binary Sensors

  • HorizonIQ Import
  • HorizonIQ Export

Sensors

  • HorizonIQ Total Cost
  • HorizonIQ Charging Cost
  • HorizonIQ Saving
  • HorizonIQ Forecast Diagnostics
  • HorizonIQ BMS State
  • HorizonIQ Trial Status

Button

  • HorizonIQ Clear Registration clears cached registration_data and triggers a refresh.

Stored Client-Side Data

Persisted in Home Assistant config entries:

  • url
  • api_key
  • battery_capacity_sensor
  • environment
  • hash
  • registration_data
  • forecast_device_id
  • forecast_device_token

The read-only Home Assistant Installation ID is read from Home Assistant core.uuid storage and displayed only as metadata.

Runtime-only values (not persisted): normalized forecast payload, periods, currency, target capacity, app trial status, authorization diagnostics, and last update status.

Request Notes

The optional app-trial headers are X-HorizonIQ-Device-Id and X-HorizonIQ-Device-Token.
